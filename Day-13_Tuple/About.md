## 🚀 stuple is a built-in data type used to store a collection of items in a single variable. It is similar to a list but with one critical difference: tuples are immutable, meaning they cannot be changed, added to, or modified once created

# Key Characteristics
# Ordered: Items have a defined order that will not change.
# Immutable: You cannot modify, add, or remove items after creation.
# Allow Duplicates: Since items are indexed, multiple items can have the same value.
# Heterogeneous: A single tuple can contain various data types, such as integers, strings, and even lists. 

# Basic Syntax
# Tuples are typically written with round brackets (), though the parentheses are technically optional if the items are separated by commas.
# Standard Creation: my_tuple = ("apple", "banana", "cherry")
# Single Item: Must include a trailing comma, or Python will treat it as a different type: single_item = ("apple",).
# Empty Tuple: empty = () or empty = tuple()

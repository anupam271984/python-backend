# 🚀 Function
# function is a reusable block of code designed to perform a specific task. Functions only execute when they are "called" or "invoked". They help organize programs into smaller, modular parts, reducing code repetition and making maintenance easier

# To define a function, use the def keyword, followed by a name, parentheses (), and a colon :. The code inside must be indented.

# In Python, arguments are the actual values you pass into a function when you call it. They allow you to send data into the function so it can perform its task dynamically.

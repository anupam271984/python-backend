# 🚀 For Loop

# In Python, a for loop is used to iterate over a sequence (like a list, a string, or a range of numbers)

# 1. The range() function (Most Common)
# If you want to run code a specific number of times, use range().
# Syntax: range(start, stop, step)

# 2. Looping Through a List or String
# You can pull items directly out of a collection.

# 3. Using break and continue
# break: Stops the loop entirely.
# continue: Skips the rest of the current turn and moves to the next one

# 4. Getting the Index with enumerate()
# If you need both the index number and the item, use enumerate().
